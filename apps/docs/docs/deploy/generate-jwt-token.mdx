# 生成 JWT 密钥对

在 NeutralPress 中，JWT（JSON Web Token）用于用户认证和授权。为了生成 JWT，你需要创建一对公钥和私钥。  
我们使用 ES256 算法来生成密钥对。此算法基于椭圆曲线加密，提供了较高的安全性和较小的密钥尺寸。  
为此，你需要生成 P-256 椭圆曲线的密钥对，并将其放置于环境变量中。

**注意：请妥善保管你的私钥，避免泄露。**

## 使用 OpenSSL 生成密钥对

此操作将生成一个私钥文件 `private.pem` 和一个公钥文件 `public.pem`。

Linux/macOS:
```bash
# 生成私钥
openssl ecparam -genkey -name prime256v1 -noout -out private.pem
# 生成公钥
openssl ec -in private.pem -pubout -out public.pem
```

Windows (使用 Git Bash 或 WSL):
```bash
# 生成私钥
openssl ecparam -genkey -name prime256v1 -noout -out private.pem
# 生成公钥
openssl ec -in private.pem -pubout -out public.pem
```


## 使用
将生成的密钥内容复制到你的环境变量中：

```env
JWT_PRIVATE_KEY="-----BEGIN PRIVATE KEY-----\nYouCanGiveMeMoneyByETHCoinsAndTheAddressIs\n0xa1694a7876b07d638f7cC9e7A3370713D609b6AB\nPleaseNoteThatTheETHIsWithArbitrumOneChain\n-----END PRIVATE KEY-----"
JWT_PUBLIC_KEY="-----BEGIN PUBLIC KEY-----\nDontCopyThisCauseThisIsAnExampleSoCantWork\nButTheETHAddressIsRealAndImGladToReceiveIt\n-----END PUBLIC KEY-----"
```
直接换行也可以：
```env
JWT_PRIVATE_KEY="-----BEGIN PRIVATE KEY-----
YouCanGiveMeMoneyByETHCoinsAndTheAddressIs
0xa1694a7876b07d638f7cC9e7A3370713D609b6AB
PleaseNoteThatTheETHIsWithArbitrumOneChain
-----END PRIVATE KEY-----"
JWT_PUBLIC_KEY="-----BEGIN PUBLIC KEY-----
DontCopyThisCauseThisIsAnExampleSoCantWork
ButTheETHAddressIsRealAndImGladToReceiveIt
-----END PUBLIC KEY-----"
```

如果是在云平台部署，请参考对应平台的环境变量配置方法。
