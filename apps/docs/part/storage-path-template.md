支持的路径模板参数:

| 参数       | 说明                 | 示例值           |
| ---------- | -------------------- | ---------------- |
| {year}     | 四位年份             | 2026             |
| {month}    | 两位月份（带前导零） | 02               |
| {day}      | 两位日期（带前导零） | 12               |
| {filename} | 完整文件名           | photo-abc123.jpg |
| {basename} | 文件名不含扩展名     | photo-abc123     |
| {ext}      | 文件扩展名（不含点） | jpg              |
