## 简介

Vercel 是 Next.js 的官方托管平台，免费套餐额度较多，同时提供 PostgreSQL 和 Redis，但瓶颈是图片优化服务较少，仅支持 1000 张照片/月。

Vercel 的云服务免费提供 `.vercel.app` 的二级域名，但因某种原因导致境内无法访问。绑定自己的域名可解决此问题。
